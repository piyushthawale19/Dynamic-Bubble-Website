## ğŸ¤ How to Contribute

We welcome contributions! Whether you're fixing a bug, improving the design, or adding new features, here's how you can help:

### 1. Fork & Clone
Fork this repository to your own GitHub account, then clone it to your local machine.

### 2. Create a Branch
Create a new branch for your specific feature or fix. This keeps the main branch clean.

```bash
git checkout -b feature/amazing-new-feature
# or
git checkout -b fix/bug-fix-name
```

### 3. Make Your Changes
Make your code changes. Please stick to the following guidelines:
*   **HTML:** Use semantic HTML5 tags.
*   **CSS:** Use the defined CSS variables in `styles.css` for colors and spacing. Avoid inline styles.
*   **Images:** Place new images in `assets/images`.

### 4. Commit & Push
Commit your changes with clear, descriptive messages.

```bash
git add .
git commit -m "Add a brief description of your changes"
git push origin feature/amazing-new-feature
```

### 5. Submit a Pull Request (PR)
Go to the original repository on GitHub and open a Pull Request. Provide a clear description of what you've done and why.
